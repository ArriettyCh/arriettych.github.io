## **课程介绍**

这是信安专业的必修课，并不在CS培养方案中。但我闲得慌，所以就选了。教学的重点是隐私计算，即如何让数据「可用不可见」——在无法看见数据明文的情况下对数据进行处理。理论课每周两节，实验课两周一次，可以不去，按时提交作业即可。
